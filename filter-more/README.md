# Filters
Implement a program that applies filters to BMPs, per the below.

Greyscale
```
$ ./filter -g infile.bmp outfile.bmp
```

Reflection
```
$ ./filter -r infile.bmp outfile.bmp
```

Blur
```
$ ./filter -b infile.bmp outfile.bmp
```

Edge Detection
```
$ ./filter -e infile.bmp outfile.bmp
```
